# 手机离线使用说明

可以做到“不和电脑在同一个网络”。推荐方式是把 `dist/` 部署成一个网页链接，同学用手机第一次联网打开一次，之后就可以离线刷题。

## 最推荐流程

1. 在你的电脑上构建：

```bash
npm run build
```

2. 把 `dist/` 上传到 Netlify、Vercel、GitHub Pages 等静态网站平台。
3. 把网站链接发给同学。
4. 同学手机第一次联网打开这个链接，等待首页正常显示。
5. 添加到手机桌面：
   - iPhone Safari：分享按钮 -> 添加到主屏幕
6. 以后没有网络也能从桌面图标打开刷题。

## 重要限制

- 第一次必须联网打开一次，因为手机需要下载网页、题库和缓存文件。
- 离线后不需要和电脑同一个 Wi-Fi，也不需要电脑开机。
- 每个人的练习记录保存在自己手机浏览器里。
- 如果清理浏览器缓存或删除网站数据，练习记录会丢失。

## 如果完全没有网络

如果手机从来不能联网打开链接，普通网页/PWA 无法可靠安装。iPhone 不能直接安装未签名的自制应用文件，只能走网页 PWA、App Store、TestFlight 或开发者签名。
