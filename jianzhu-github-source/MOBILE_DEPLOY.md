# 手机分享部署说明

这个项目是纯前端应用，不需要服务器数据库。部署后同学可以直接用手机浏览器打开链接刷题，练习记录会保存在各自手机浏览器的 IndexedDB 中。

## 推荐方式：静态网站托管

构建命令：

```bash
npm install
npm run build
```

构建完成后，把 `dist/` 文件夹部署到任意静态网站平台即可，例如：

- Netlify：拖拽 `dist/` 文件夹发布。
- Vercel：导入项目，构建命令填 `npm run build`，输出目录填 `dist`。
- GitHub Pages：上传项目后用 GitHub Actions 构建发布。

部署成功后，把生成的网址发给同学即可。

## 临时方式：同一 Wi-Fi 手机访问

如果只是课堂或宿舍临时使用，电脑和手机连同一个 Wi-Fi 后，电脑运行：

```bash
npm run dev
```

终端会显示类似：

```text
Network: http://192.168.x.x:5173/
```

手机浏览器打开这个 `Network` 地址即可。这个方式要求你的电脑一直开着，且手机和电脑在同一个网络。

## 手机安装到桌面

网页打开后：

- iPhone Safari：点击分享按钮，选择“添加到主屏幕”。

安装后看起来会像一个轻量 App。

## 注意

- 每个同学的练习记录只保存在自己的手机浏览器里，不会互相同步。
- 如果同学清理浏览器数据，学习记录会被清空。
- 题库已经内置在网页里，正常访问不需要再导入 Word 或 JSON。
